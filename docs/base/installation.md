Installation
============

# Installation of pytorch

For installing *pytorch* follow [Quick Start Locally](https://pytorch.org/) for your config.

# Installation of torchforce
- Download files:

```
git clone https://github.com/french-ai/reinforcement.git
```

In root repository :

* for use:
```
pip install .
```

* for dev:
```
pip install ".[dev]" .
```
